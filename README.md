# Purity Komer - Week 03 Command Line & Git Project

## Project Overview

This week I learned and practiced essential command line and Git/GitHub skills.  
These include working with terminal commands (PowerShell/Linux), creating project structures, managing directories, and using Git for version control.

I also learned how to collaborate using GitHub, work with branches, resolve merge conflicts, and create a `.gitignore` file to keep projects clean.

## Features

- Terminal navigation using CLI (pwd, ls, cd)
- File and directory operations using terminal commands
- Git initialization and commit history tracking
- Branching and merging workflows
- Merge conflict handling
- GitHub remote repository integration
- `.gitignore` configuration to exclude unnecessary files


## Technologies Used

- Command Line (PowerShell)
- Git & GitHub
